**Date:** 2026-03-23

## Executive Summary

| Feature | Barron's 2024 | Princeton 2024 |
|---|---|---|
| Has a diagnostic test? | **No** | **Yes** (Practice Test 1 doubles as diagnostic) |
| Number of diagnostic MC questions | N/A | **55** |
| Maps questions to CED Units? | No | **Yes** (Unit 1-9) |
| Maps questions to CED Topics? | No | **No** (Unit-level only, not Topic 1.1/1.2 etc.) |
| Self-assessment guide? | No | **Yes** (per-unit score tally with study plan guidance) |
| Practice test answer keys map to units? | No | Only Practice Test 1 |
| Total practice tests | 2 | 4 (in-book) + 2 online |

**Bottom line:** The prep books tell you "Question 5 tests Unit 2" but they do NOT tell you "Question 5 tests Topic 2.3." Mapping to CED topic-level granularity (e.g., Topic 1.1, 2.3, 5.1) must be done manually.

---

## Barron's 2024: No Diagnostic Test

Barron's does not include a diagnostic test. The book structure is:
- **Section Zero:** Foundations (pre-1200 background)
- **Sections 1-4:** Content review organized by AP time periods, with thematic chapters (Governance, Culture, Technology, Economics, Social, Environment)
- **Fact-Check Review Quizzes:** ~5-10 questions per chapter, but with NO unit/topic mapping in the answer keys
- **2 Practice Tests:** Full 55-question MC sections + FRQs, with detailed answer explanations but NO unit/topic mapping

The Barron's answer explanations are pedagogically rich (they explain why each wrong answer is wrong) but provide zero structural metadata about which CED unit or topic each question tests.

---

## Princeton 2024: Diagnostic Test with Unit-Level Mapping

Princeton's Practice Test 1 serves as the diagnostic. It has a dedicated "Diagnostic Answer Key" section with a three-step process:
1. Check answers against a table that maps each question to a **CED Unit**
2. Tally correct answers per unit to get a percentage score
3. Use results to customize a study plan (focus on lowest-scoring units)

## Gap Analysis

### Coverage Gaps in Princeton Diagnostic
1. **Unit 9 (Globalization): ZERO MC questions.** This is a significant gap. Unit 9 is only covered by SAQ Q4.
2. **Unit 5 (Revolutions): Only 3 questions (5.5%).** CED weight is 12-15%. Significantly under-tested.
3. **Unit 4 (Transoceanic Interconnections): Only 4 questions (7.3%).** CED weight is 12-15%.

### Over-representation
1. **Unit 8 (Cold War and Decolonization): 12 questions (21.8%).** CED weight is 12-15%. Nearly double.
2. **Unit 1 (Global Tapestry): 9 questions (16.4%).** CED weight is 8-10%.

### No Topic-Level Granularity
Neither book maps to CED topic numbers (e.g., Topic 1.1, 1.2, 2.1). The finest granularity available is **CED Unit** (Unit 1-9). To create a topic-level diagnostic mapping (e.g., "Question 5 tests Topic 2.3: Exchange in the Indian Ocean"), you would need to:
- Read each question and its stimulus
- Match the content to a specific CED topic based on the topic descriptions in the CED
- This is a manual process -- the prep books do not do this for you

## Barron's Practice Tests: No Mapping At All

Both Barron's practice tests (55 MC questions each = 110 total) provide detailed answer explanations but zero structural metadata:
- No unit mapping
- No topic mapping
- No self-assessment guide
- No diagnostic scoring rubric

The explanations do reference AP-relevant concepts (e.g., "a key AP topic") but never label them with unit/topic numbers.

Barron's also has ~150 "Fact-Check Review Quiz" questions across 25 chapters, organized by thematic category and time period. These could theoretically be reverse-mapped to CED units based on chapter placement (e.g., Chapter 1 = Governance 1200-1450 = Units 1-2), but this mapping is not provided by the book.

---

## Recommendation

**Use the Princeton diagnostic** as the starting point. It is the only diagnostic available with any unit-level mapping.

However, be aware of its limitations:
1. **Unit-level only, not topic-level.** You will need to manually map each question to a CED topic (e.g., Topic 1.1, 2.3) by reading the question content.
2. **Unit 9 (Globalization) has zero MC coverage.** You will need to supplement with questions from other sources.
3. **Unit 5 (Revolutions) and Unit 4 (Transoceanic Interconnections) are under-represented.** Consider weighting these more heavily in supplementary assessments.
4. **Unit 8 is heavily over-represented.** The diagnostic will give an inflated sense of how well students know Cold War material relative to other units.

For a production-quality diagnostic that maps to CED topics: **we have to build the topic-level mapping ourselves.** No commercially available prep book provides it.
