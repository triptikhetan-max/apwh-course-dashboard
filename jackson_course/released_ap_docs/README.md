## DBQ Documents (for HIPP Practice)

Every FRQ set listed above contains a DBQ section with primary/secondary source documents. These are the key files for HIPP (Historical context, Intended audience, Purpose, Point of view) analysis practice:

- All 11 FRQ files contain DBQ documents suitable for HIPP practice
- DBQs typically include 5-7 documents (text excerpts, images, maps, charts, data tables)
- The scoring guidelines contain the expected HIPP analysis points for each document

## SAQ Stimuli

Every FRQ set contains Short Answer Questions (SAQs). SAQs 1-2 include stimulus material (primary/secondary source excerpts, images, charts, maps, or data). SAQs 3-4 are typically non-stimulus.
